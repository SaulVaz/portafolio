Ago 2024 — Sep 2024 Proyecto académico
Titanic — Machine Learning from Disaster — Tec de Monterrey

Modelo de machine learning para predecir la supervivencia de pasajeros del Titanic. Evaluamos múltiples algoritmos (Árbol de Decisión, Red Neuronal, Regresión Logística y Random Forest), logrando la mayor precisión con Random Forest. Trabajé en la limpieza, transformación y feature engineering del dataset para garantizar la calidad del modelo.

// Por qué estas tecnologías

La decisión de usar Random Forest sobre los demás modelos no fue arbitraria: tras comparar métricas de accuracy, precisión y recall en cada algoritmo, Random Forest generalizó mejor por su naturaleza de ensemble, reduciendo el overfitting que sí presentó el Árbol de Decisión individual. Scikit-Learn por su API unificada que nos permitió cambiar de modelo con mínimo refactoring durante la comparación.

Python Scikit-Learn Random Forest Pandas Google Colab ML

Oct 2025 — Feb 2026 Proyecto personal
GitHub ↗
GymApp — Aplicación Móvil Full Stack — Proyecto personal

Aplicación móvil multiplataforma completa para gestión de rutinas de gimnasio. Construí una REST API con más de 20 endpoints que maneja autenticación JWT con roles, CRUD completo de rutinas, catálogo de 19 ejercicios por grupos musculares, sistema de favoritos y registro de sesiones con historial detallado de progreso. El frontend móvil incluye autenticación persistente, entrenamiento en tiempo real con registro de peso/reps y estadísticas de progreso. Base de datos modelada con 8 tablas relacionales en PostgreSQL.

// Por qué estas tecnologías

React Native + Expo para desarrollo móvil multiplataforma desde una sola base de código — iOS y Android sin duplicar lógica. PostgreSQL con Prisma fue clave: la naturaleza relacional de los datos (usuarios → rutinas → ejercicios → sesiones → sets) exigía un esquema tipado y consultas JOIN eficientes, imposibles de lograr limpiamente con una BD NoSQL. JWT con bcrypt para autenticación stateless que escala sin sesiones en servidor. Prisma ORM para type-safety en las queries y migraciones controladas del schema.

React Native Expo Node.js Express PostgreSQL Prisma JWT TypeScript Axios
